# Prompt2Pic – Overview

## Description

Prompt2Pic is a personal AI-powered creative assistant that I built to simplify my own workflow of generating AI images. Previously, I had to manually write prompts and generate each image one by one, which was repetitive and time-consuming. To solve this, I created Prompt2Pic to automate the entire process in batch — from idea to final images.

## Key Features

* **Idea to Prompt Conversion**

  * Enter a simple concept or idea.
  * Automatically generates 6 creative and diverse prompts.

* **Batch Image Generation**

  * Sends all prompts to a free AI image generation API.
  * Generates multiple images in one go without manual repetition.

* **Automated Workflow**

  * Eliminates the need to copy-paste prompts and generate images individually.
  * Designed to save time and reduce effort.

* **Image Preview and Download**

  * View all generated images directly in the interface.
  * Download images individually as needed.

* **Local Storage with Serial Naming**

  * Automatically saves images locally.
  * Uses serial numbering for organized file management.

## User Workflow

1. Enter a concept or idea.
2. Instantly receive 6 AI-generated prompts.
3. Generate all images in batch automatically.
4. Preview the generated images.
5. Download or access saved images locally.

## Purpose

This project was built from a personal need to automate repetitive tasks in AI image generation. It focuses on improving efficiency, reducing manual work, and making the creative process faster and more streamlined.

## Use Cases

* Social media content creation
* Digital art generation
* Idea visualization
* Creative experimentation
* Rapid content production
